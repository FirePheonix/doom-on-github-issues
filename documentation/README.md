# Documentation (React 19)

This folder now runs as a React 19 + Vite docs app.

## Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deploy the `documentation/` project root on Vercel with:

- Install command: `npm install`
- Build command: `npm run build`
- Output directory: `dist`
