Static release notes for the Doom-on-GitHub-Issues project, ready for Vercel.

Point a Vercel project at documentation/ and this site will deploy as-is, with no build step. If you deploy the repo root instead, the top-level redirect sends visitors here.

Source code: https://github.com/FirePheonix/doom-on-github-issues

Fastest path: Vercel project root = documentation/. The docs shell, sidebar links, and hash-based document navigation all work from static hosting.
